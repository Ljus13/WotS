# WotS

Lightweight static repository showcasing basic HTML and CSS snippets.  All styles live in `css/wots.css` and pages are under `html/`.  No build tools or Node dependencies are required—simply open the HTML files in a browser.

Examples:

- `index.html` – home page with navigation
- `html/example1.html` – typography demo
- `html/example2.html` – layout/card demo

Feel free to add new HTML pages and style them with CSS.
